# Apple-style Static Shopping Website

This is a static Apple-style shopping website using only:

- HTML
- CSS
- JavaScript

No React, no Vite, no build step, no CDN, and no external animation library.

## Run locally

Open `index.html` directly in a browser, or use VS Code Live Server.

## Main files

- `index.html` — page structure
- `styles.css` — Apple-style layout, responsive design, lightweight animations
- `script.js` — product rendering, filters, product details modal, cart drawer, FAQ accordion, carousel controls
- `assets/products/` — optimized WebP product images used by the website
- `assets/originals/` — original uploaded PNG product images, preserved for editing

## Production notes

This is a front-end demo only. Do not collect real payment data unless you add a secure backend, payment gateway integration, validation, CSRF protection, rate limiting, logging, monitoring, and proper error handling.
